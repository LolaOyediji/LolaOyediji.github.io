Inputs


Type Tag Description
Text <type="text"> Renders a text field that users can type into.
Number <type="number"> Renders a field that users can type numbers into.
Password <type="password"> Renders a text field that users can type into.
Range <type="range"> Creates a slider with a set minimum and maximum value.
Dropdown <select> <option> Allows users to choose one option from an organized list.
Datalist <type="text"> <list="id"> Creates a text field that users can type into and filter options from the datalist.
Radio <type="radio"> Renders buttons for options where only one can be selected.
Checkbox <type="checkbox"> Renders boxes to select multiple options.
